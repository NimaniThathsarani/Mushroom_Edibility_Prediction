This project focuses on predicting mushroom edibility (edible vs poisonous) using the UCI Mushroom Dataset. The aim is to apply data cleaning, preprocessing, and exploratory data analysis to prepare the dataset for building reliable ML model.

Problem Domain: Agriculture/Food Safety
Impact: Automating mushroom classification reduces health risks, supports foragers and researchers, and can be integrated into mobile identification tools.

Dataset Details:

Dataset Name: UCI Secondary Mushroom Dataset
Source: UCI Machine Learning Repository
Link: https://archive.ics.uci.edu/dataset/848/secondary+mushroom+dataset
Dataset Type: Structured Tabular Data
Size: 61,029 samples
Features: 20 categorical attributes
Target: Edible(e)/Poisonous(p)
